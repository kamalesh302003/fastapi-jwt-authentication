#FastAPI JWT Authentication System
A secure authentication system built using FastAPI and JWT token-based authentication.  
Implements OAuth2 password flow with bcrypt password hashing for user verification.  
Includes login functionality, token generation, and protected API routes.

##Features
- FastAPI framework implementation
- JWT (JSON Web Token) authentication
- OAuth2 password flow
- Password hashing using bcrypt
- Secure login endpoint
- Protected routes with token verification
- Token expiration support

##Technologies Used
- Python
- FastAPI
- JWT (python-jose)
- Passlib
- OAuth2
- Uvicorn
